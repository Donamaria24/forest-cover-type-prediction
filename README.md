**Forest Cover Type Prediction**

**Objective**
Predict the type of forest cover for a 30m x 30m patch of land using cartographic variables.

**Dataset**
Data collected from the Roosevelt National Forest, Colorado.

**Forest Cover Types:**
1. Spruce/Fir
2. Lodgepole Pine
3. Ponderosa Pine
4. Cottonwood/Willow
5. Aspen
6. Douglas-fir
7. Krummholz

**Files**
- `Forest_Cover_Type_Prediction.ipynb` – Model training & evaluation
- `Forest Cover Type Prediction.pdf` – Problem statement
- `train.csv` – Dataset (if included)

**Technologies Used**
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
